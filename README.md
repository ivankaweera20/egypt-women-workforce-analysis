# Empowering Data Final Project – Women Workforce Analysis (Egypt)

## Overview
This Python project analyzes and visualizes trends in women's workforce participation in Egypt from 1990 to 2015 using real-world CSV data. The program uses pandas for data processing and matplotlib for data visualization.

## Features
- Reads CSV data using pandas
- Filters data for Egypt
- Generates a scatter plot of women working (%) over time
- Automatically saves the graph as a PNG file
- Includes research context, characters, and analysis question

## Files Included
- main.py → Main program code
- livwell135.csv → Dataset used for analysis
- GWCutilities.py → Utility functions for formatting output
- women_working_scatter.png → Generated graph output

## How to Run
1. Make sure Python is installed
2. Install required libraries:
   pip install pandas matplotlib
3. Run the program:
   python main.py

## Output
The program automatically generates and saves:
women_working_scatter.png
